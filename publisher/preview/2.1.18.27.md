# Publisher Preview Version 2.1.18.27

**Release Date:** 2023-11-07

## Overview

This preview release adds support for detecting software with translated DisplayName, implements Publish Now for Custom Apps, and fixes issues with dependencies and delayed ConfigMgr app publishing.

---

## Features

New capabilities and functionality added in this release.

- **DisplayName Translation** - Support detecting software that translates DisplayName
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1335)*
- **Publish Now for Custom Apps** - Implement Publish Now for Custom Apps

---

## Improvements

Enhancements to existing functionality.

- **Tab Navigation** - Improve some popup notifications to direct the user to the correct tab.
- **Service Shutdown** - Improve cleanup during service shutdown.

---

## Fixes

Bug fixes and issue resolutions.

- **WSUS Certificate Link** - Fixed a dead 'More Info' link for WSUS certificate management.
- **Dependency Preservation** - Fixed a bug causing dependencies to be removed from an Intune Win32 application when republishing.
- **Delayed Publishing** - Fixed a bug where publish now and delayed ConfigMgr apps did not work as expected.
