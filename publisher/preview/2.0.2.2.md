# Publisher Preview Version 2.0.2.2

**Release Date:** 2021-04-19

## Overview

This preview release enables defining ConfigMgr security scopes directly in the Publisher service, improves scan wizard export options, and fixes several bugs related to proxy configuration and reporting.

---

## Features

New capabilities and functionality added in this release.

- **ConfigMgr Scopes** - Define ConfigMgr scopes inside the Publisher service
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-962)*
  - Note: This requires updated permissions. The 'Import Role' option in the Publisher will import the role with proper permissions or you can refer to this [article](https://patchmypc.com/permissions-required-in-sccm-for-base-installation-packages-from-patch-my-pc).

---

## Improvements

Enhancements to existing functionality.

- **Scan Wizard Export** - The Intune and ConfigMgr scan wizard 'Export' buttons now prompt for whether the filter should be applied to the export.
- **Timestamping Handling** - Improve how Timestamping is handled in some scenarios.

---

## Fixes

Bug fixes and issue resolutions.

- **Intune Proxy** - Connections to Intune may not respect the proxy configuration set in the Publisher.
- **Package Details** - 'Show Package Detailsâ€¦' right-click option would not load as expected.
- **CVE Wizard Loading** - CVE Wizard would not load as expected.
- **SSRS Compliance** - SSRS dashboards would report a negative % for compliance in some scenarios. The reports can be reinstalled if you are affected by following the same process as the initial install which will overwrite the reports.
