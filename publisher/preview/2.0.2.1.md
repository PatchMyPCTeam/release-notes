# Publisher Preview Version 2.0.2.1

**Release Date:** 2021-04-14

## Overview

This preview release adds usage statistics display, changes license input to show only the 20-character ID, improves Intune application deletion speed with batch calls, and moves WSUS options to a more logical location.

---

## Features

New capabilities and functionality added in this release.

- **Usage Statistics** - Added the Usage Statistics group in the General tab that will show usage statitics
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1343)*
- **License Input Simplification** - Changed the license input to display only the 20 character license id and not the full license URL

---

## Improvements

Enhancements to existing functionality.

- **Batch Deletion Speed** - Improved the speed of Intune application deletion by using batch calls to Microsoft Graph
- **WSUS Options Location** - Options related to WSUS have been moved from the Advanced tab to the Options button in the Updates tab

---

## Fixes

Bug fixes and issue resolutions.

- **Classification Field** - The classification field in the Intune Apps Manager is not populated for Updates
- **Update Revision** - Update revision doesn't take account of republished updates
