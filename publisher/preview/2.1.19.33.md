# Publisher Preview Version 2.1.19.33

**Release Date:** 2024-02-15

## Overview

This preview release reduces memory usage during .intunewin uploads, improves Publisher startup when ConfigApi is unavailable, and fixes several bugs related to custom apps and Azure uploads.

---

## Improvements

Enhancements to existing functionality.

- **Memory Optimization** - Use less memory when uploading .intunewin files to Intune.
- **ConfigApi Availability** - Improve the user experience when the ConfigApi is not available
  - The UI can now start without the ConfigApi being available. A popup message will still appear indicating it cannot be reached.
  - When this is not available, the cloud features of the Publisher will be disabled.

---

## Fixes

Bug fixes and issue resolutions.

- **Right-Click State** - Fixed a bug that caused some right-click options to not clear their state as expected for custom apps.
- **Apply Button** - Editing Manage Conflicting Process options did not light up the Apply button.
- **Azure Upload Retry** - Fixed a bug that caused the Publisher to fail to retry when uploading chunks to Azure.
- **Custom App Hash** - Fixed a bug where a custom application download may compare hash against an older version of the custom application's hash.
