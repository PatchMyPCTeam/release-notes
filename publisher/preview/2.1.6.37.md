# Publisher Preview Version 2.1.6.37

**Release Date:** 2022-09-28

## Overview

This preview release adds missing tooltips, improves button clarity, and fixes a critical bug where All Devices and All Users assignments could be created with incorrect intent.

---

## Improvements

Enhancements to existing functionality.

- **Scan Wizard Tooltip** - Added a missing tooltip to the ConfigMgr scan wizard button in the ConfigMgr tab.
- **Copy Button Clarity** - Added clarity to the 'copy' button tooltips in the Publisher.
- **MCP Whitespace Parsing** - Fixed whitespace handling for [Manage Conflicting Processes](https://patchmypc.com/manage-conflicting-processes-when-updating-third-party-applications) Organization Name.
- **Settings Backup** - Fixed old setting backup renaming issue.

---

## Fixes

Bug fixes and issue resolutions.

- **Assignment Intent** - Fixed a bug where All Devices and All Users assignments created in the Manage Assignments UI could be created with the wrong intent.
  - Note: If you created any new assignments in the Manage Assignment UI while on build 2.1.6.35 please check that they are correctly configured.
  - All Devices assignments default to 'Uninstall' intent.
  - All Users assignments default to 'Required' intent.
