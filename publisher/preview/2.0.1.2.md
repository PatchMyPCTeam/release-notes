# Publisher Preview 2.0.1.2 - 2021-03-11

## Overview

This preview release includes features, improvements, and fixes for the Patch My PC Publisher.

## Features

- You can now remove assignments in Intune with a right-click option

## Improvements

- Improved logging around the Azure application authentication process
- You can now copy the detection method PowerShell script with a right-click option

## Fixes

- Fixed an issue where adding multiple assignments to an application would only reflect the last added assignment on an application update if they had the same name
- We no longer try to update an application when a newer update version is superseded by a previous version
- Fixed an issue where an application uninstall may not work when there are multiple Install keys/uninstall strings detected
