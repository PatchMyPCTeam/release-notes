# Publisher Preview 1.9.3.3 - 2020-11-03

## Overview

This preview release includes features, improvements, and fixes for the Patch My PC Publisher.

## Features

- [ConfigMgr Database Scan Wizard updated to include 'TargetVersion' as a column. Based on Aha Idea: PATCHMYPC-I-836](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-836)
- [ConfigMgr Database Scan Wizard updated to include the application count on results. Based on Aha Idea: PATCHMYPC-I-836](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-836)

## Improvements

- ConfigMgr Database Scan Wizard can now be closed via ESC key in addition to cancel button
- Improve logging around 'Test Connection' when publishing via HTTPS to WS US through a proxy

## Fixes

- Fixed some typos in the ConfigMgr database scan wizard
