# Publisher Preview Version 2.1.10.9

**Release Date:** 2022-12-06

## Overview

This preview release improves alerting for missing files, enhances email report links to release notes, and fixes several bugs related to scripts, uninstall arguments, and UI components.

---

## Improvements

Enhancements to existing functionality.

- **Missing Files Alerts** - Improved the content of alerts when additional files or folders are missing when Publishing a product
- **SMS Provider Button** - The 'Configure SMS Provider connection' button is no longer highlighted if unconfigured in WSUS Standalone Mode.
- **SMTP Logging** - Improved the logging for SMTP initialization and error handling.
- **Email Report Links** - All titles in the email report now link to release notes if available.

---

## Fixes

Bug fixes and issue resolutions.

- **Script Addition** - Fixed a bug where the Publisher would not add a PMPC-defined script to an existing product.
- **MSI Uninstall Arguments** - Fixed a bug where Scriptrunner did not append the provided Silent Uninstall Arguments to MSI uninstalls.
- **ListView Filter** - Fixed a bug where some UI listviews had a broken filter.
- **MCP Window Position** - Fixed a bug where Manage Conflicting Process window may be offset from the bottom right corner.
