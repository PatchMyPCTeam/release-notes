# Publisher Version 2.1.9 - Production

**Release Date:** 2022-10-14

## Overview

This maintenance release addresses several bugs including Intune Standalone Mode tab visibility, scriptrunner registry parsing issues, and dynamic assignment matching problems.

---

## Fixes

Bug fixes and issue resolutions.

- **Intune Standalone Mode** - Fixed a bug where selecting Intune Standalone Mode during the initial installation of the Publisher would not hide the Updates and ConfigMgr Apps tabs.
- **Settings Backup Rename** - Fixed a bug that caused a failure to rename legacy settings backup CAB files.
- **Registry Parsing** - Fixed a bug where Scriptrunner would fail to parse the registry when searching for Uninstall keys if there is an empty string in the SystemComponent property.
- **Dynamic Assignments** - Fixed a bug where [Intune Dynamic Assignments](https://patchmypc.com/manage-dynamic-assignments) may unexpectedly match against all selected updates.
- **Republish Flag** - Fixed a bug where the '[Republishâ€¦](https://patchmypc.com/custom-options-available-for-third-party-updates-and-applications#republish-updates)' option is not unchecked on Intune products if the UI is left open while the synchronization runs.
