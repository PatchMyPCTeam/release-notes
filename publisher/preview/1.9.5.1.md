# Publisher Preview 1.9.5.1 - 2020-11-09

## Overview

This preview release includes features, improvements, and fixes for the Patch My PC Publisher.

## Features

- [Administrators can now view the configuration of packages via the Package Details right-click option, and even export the settings via the Import/Export buttons at the bottom of the window. Based on Aha Idea: PATCHMYPC-I-788](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-788)

## Improvements

- [Intune Scan Wizard has been improved to allow counting only Required or Available assignments and no longer counts Assignment overrides for packages](https://patchmypc.com/intune-application-creation-options#intune-application-scan-wizard-improvements)

## Fixes

- Fixed an issue where the package details feature could not be opened for some specific update revisions
