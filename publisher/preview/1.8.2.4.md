# Publisher Preview 1.8.2.4 - 2020-06-26

## Overview

This preview release includes features, improvements, and fixes for the Patch My PC Publisher.

## Features

- [Intune Application Scan Wizard. This new tool will scan all applications in your Intune tenant, match them with applications available within the Patch My PC catalog, and allow you to easily enable them within the Publisher. Based on Aha Idea: PATCHMYPC-I-477](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-477)

## Improvements

- Progress percentages are now displayed while downloading installers and uploading content to Intune
- [Improved "Manage Application" features for Intune. There is now a search box, and a "View in Intune" button which opens the application in Microsoft Endpoint Manager admin center](https://patchmypc.com/intune-application-creation-options#manage-intune-applications)

## Fixes

- Fixed an issue where the icon for an application may not be displayed if the icon was an SVG image
