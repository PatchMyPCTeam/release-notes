# Publisher Preview Version 2.1.22.7

**Release Date:** 2024-04-30

## Overview

This preview release adds support for 'Allow available uninstall' in Intune, improves detection regex handling by storing it in base64 format, and fixes issues with large custom application files.

---

## Features

New capabilities and functionality added in this release.

- **Available Uninstall Support** - Support for 'Allow available uninstall' in Intune
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-3213)*

---

## Improvements

Enhancements to existing functionality.

- **Detection Regex Improvements** - Improve our regex usage in the detection of applications published by Patch My PC.
  - The regex string is now stored in base64 in the PowerShell script to prevent Intune from clobbering UTF8 characters.

---

## Fixes

Bug fixes and issue resolutions.

- **Large File Processing** - Fixed a bug that caused custom applications with a main file larger than 2GB to fail to process.
