# Publisher Preview Version 2.1.14.11

**Release Date:** 2023-05-17

## Overview

This preview release updates graph API calls to use smaller page sizes, reducing the likelihood of receiving 429 or 503 response errors from Microsoft Graph.

---

## Improvements

Enhancements to existing functionality.

- **Graph Page Size** - Updated some graph calls to use a smaller page size. This reduces the chance of receiving a 429 or 503 response from graph.
