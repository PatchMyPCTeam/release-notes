# Publisher Version 2.1.15 - Production

**Release Date:** 2023-06-14

## Overview

This release adds support for .cmd files in pre/post-scripts, improves Graph API call efficiency to reduce 429 responses, and fixes supersedence relationship issues with ConfigMgr applications.

---

## Features

New capabilities and functionality added in this release.

- **CMD File Support** - Add support for .cmd files in pre/post-scripts.
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-2745)*

---

## Improvements

Enhancements to existing functionality.

- **Package Info Update ID Column** - The Update ID is now an available column in the [Show package info](https://patchmypc.com/custom-options-available-for-third-party-updates-and-applications#PackageInfo) tool.
- **Graph API Page Size** - Updated some graph calls to use a smaller page size. This reduces the chance of receiving a 429 or 503 response from graph.
  - The page size is configurable from 20-999 in the Advanced tab of the Publisher.
- **Automatic Product Sorting** - The product treeviews are now sorted by Vendor and then Product Name automatically.

---

## Fixes

Bug fixes and issue resolutions.

- **Supersedence Relationship** - Fixed a bug where a ConfigMgr application's supersedence relationship was lost when the application was upgraded in place.
- **Email Report Sorting** - Fixed a bug where the sorting of products in the email report was in reverse alphabetical order.
