# Publisher Preview Version 2.1.9.7

**Release Date:** 2022-10-21

## Overview

This preview release adds a Superseded filter to the Modify Updates Wizard, adds customization for parallel intunewin upload threads, and improves proxy configuration logging.

---

## Improvements

Enhancements to existing functionality.

- **Superseded Filter** - Added a filter for Superseded to the [Modify Updates Wizard](https://patchmypc.com/modify-published-third-party-updates-wizard).
- **Log Level Adjustments** - Adjusted some log levels and log text to be clearer.
- **Proxy Error Logging** - Added additional logging when the proxy settings are loaded in the event of a failure.
- **Upload Thread Customization** - Added the option to customize the number of parallel threads to use when performing an upload of an intunewin file to Intune.
