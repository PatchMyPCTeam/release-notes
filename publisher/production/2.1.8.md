# Publisher Version 2.1.8 - Production

**Release Date:** 2022-10-10

## Overview

This critical hotfix release addresses a bug that caused ConfigMgr applications to fail installation with error 0x80070057 and improves .intunewin file creation speed.

---

## Improvements

Enhancements to existing functionality.

- **Intunewin Creation Speed** - Improve speed of creating .intunewin files for Intune package creation.

---

## Fixes

Bug fixes and issue resolutions.

- **ConfigMgr Installation Error** - Fixed a bug where ConfigMgr applications would fail to install with error code 0x80070057 or "The parameter is incorrect" during the installation.
  - If an existing application has this problem in your environment you will need to [republish](https://patchmypc.com/custom-options-available-for-third-party-updates-and-applications#republish-updates) the affected application.
  - The trigger for this bug is having the ConfigMgr option set to "[Update existing application's metadataâ€¦](https://patchmypc.com/base-install-update-options-explained#topic1)"
  - This bug was introduced in preview version 2.1.6.35 and carried through to production version 2.1.7.0.
