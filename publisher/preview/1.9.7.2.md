# Publisher Preview 1.9.7.2 - 2020-12-07

## Overview

This preview release includes features and improvements for the Patch My PC Publisher.

## Features

- [ScriptRunner now has a new custom variable, %CurrentDir%. This variable may need to be put in double-quotes or the entire parameter may need to be in double quotes depending on the application. Examples: Config="%CurrentDir%\Config.ini" or "Config=%CurrentDir%\Config.ini". Based on Aha Idea: PATCHMYPC-I-985](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-985)

## Improvements

- The Intune Application Manager now has a button for 'Manage Assignments'
- Improve how ScriptRunner handles version parts that exceed the 32 bit signed integer max
- [Improve CVE Import Wizard CVE-ID matching](https://patchmypc.com/how-the-cve-import-wizard-works-for-matching-cve-ids)
- Improve logging associated with delayed application publishing
- [The SSRS report dashboards now include a parameter for Deployed](https://patchmypc.com/free-software-update-compliance-dashboard-reports-for-microsoft-sccm)
