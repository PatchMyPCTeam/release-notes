# Publisher Version 2.1.18 - Production

**Release Date:** 2023-10-16

## Overview

This release introduces the ability to sync only selected apps/updates during a sync operation, adds Delivery Optimization management for Intune assignments, and provides bulk operation capabilities in the Intune Application Manager Utility. A new download engine improves reliability across all components.

---

## Features

New capabilities and functionality added in this release.

- **Selective Sync** - Sync only selected apps/updates during a sync.
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-468)*
- **Delivery Optimization Management** - Allow the management of Delivery Optimization configuration for Intune assignments.
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-2071)*
  - Note: Existing DO configuration on an assignment will also carry forward now when a new version of the software is Published.
- **Intune App Manager Bulk Options** - The [Intune Application Manager Utility](https://patchmypc.com/intune-application-manager-utility) now has some multi-select bulk options.

## Fixes

Bug fixes and issue resolutions.
