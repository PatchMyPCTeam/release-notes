# Publisher Preview 1.9.7.3 - 2020-12-11

## Overview

This preview release includes features and improvements for the Patch My PC Publisher.

## Features

- [Manage categories for Configuration Manager applications. User Categories: Viewable to users in Software Center. Admin Categories: Viewable to administrators in the Configuration Manager Console. Based on Aha Idea: PATCHMYPC-I-780](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-780)

## Improvements

- During a Publisher sync, the Settings.xml will be automatically updated according to SupportProducts. This improves the experiences when Patch My PC makes metadata changes that impact some of the application configurations such as detection
