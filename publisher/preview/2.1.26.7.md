# Publisher Preview Version 2.1.26.7

**Release Date:** 2024-08-29

## Overview

This preview release adds support for additional download headers and improves Intune detection to handle registry property checks for non-existent properties.

---

## Improvements

Enhancements to existing functionality.

- **Download Headers Support** - Add support for passing additional headers when downloading binaries. This is metadata maintained by Patch My PC.

---

## Fixes

Bug fixes and issue resolutions.

- **Registry Detection** - Update Intune detection to account for registry detection checking for a nonexistent property. We now allow a NotEquals check for a property that might not exist. Previously, this would cause an unhandled exception.
