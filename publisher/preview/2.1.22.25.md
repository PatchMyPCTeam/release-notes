# Publisher Preview Version 2.1.22.25

**Release Date:** 2024-05-21

## Overview

This preview release implements coexistence features to alert customers when products are already managed by Intune Apps for Patch My PC Cloud, and fixes bugs related to dependency relationships and retention.

---

## Features

New capabilities and functionality added in this release.

- **Coexistence Implementation** - Implement coexistence
  - Coexistence ensures the customer is aware when a product is already managed by Intune Apps for Patch My PC Cloud.

---

## Fixes

Bug fixes and issue resolutions.

- **Dependency Circular Reference** - Fixed a bug that caused a circular reference when processing Intune dependency relationships with greater than 2 layers.
- **Retention Download Failure** - Fixed a bug that caused ConfigMgr applications to be copied for retention even if the current version fails to download.
- **Wildcard Detection** - Ensure proper wildcard support for ? in detection.
