# Publisher Preview 1.9.6.2 - 2020-11-20

## Overview

This preview release includes features and improvements for the Patch My PC Publisher.

## Features

- [Manage Intune categories for created and updated Intune applications and updates. Note: Currently, existing applications/updates in Intune will not have their categories modified. The categories will only be modified when a new Win32 application is published such as when new software is selected, or a new version is released. Based on Aha Idea: PATCHMYPC-I965](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-965)
- Clicking a 'digest' in the Modify Published Updates Wizard, or the Package Details wizard will now open the respective VirusTotal page

## Improvements

- Improved error messages when failing to publish the content library for a package
