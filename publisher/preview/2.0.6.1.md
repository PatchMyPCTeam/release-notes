# Publisher Preview Version 2.0.6.1

**Release Date:** 2021-06-24

## Overview

This preview release allows scheduling of the Modify Updates Wizard for update cleanup, fills disk space requirements for Intune apps, and improves the Intune App Manager filtering.

---

## Features

New capabilities and functionality added in this release.

- **Scheduled Update Cleanup** - Allow the scheduling of the Modify Updates Wizard for Update Cleanup (unreferencedpackagefolders)
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-797)*
- **Disk Space Required** - Fill out 'Disk Space Required' for Intune apps.
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1466)*
- **App Manager Filtering** - Intune app manager will filter by PMPC published apps by default, providing a drop-down to select non-PMPC or all apps.

---

## Fixes

Bug fixes and issue resolutions.

- **Culture Compatibility** - Further improvements to CM log format for culture compatibility.
