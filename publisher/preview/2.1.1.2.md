# Publisher Preview Version 2.1.1.2

**Release Date:** 2021-12-06

## Overview

This preview release updates to the latest Patch My PC logo and fixes several bugs related to republishing, tab settings, webhook imports, and timeout configuration.

---

## Improvements

Enhancements to existing functionality.

- **Logo Update** - Updated to the latest Patch My PC logo.

---

## Fixes

Bug fixes and issue resolutions.

- **Republish Duplication** - Fixed a bug where the Republish feature may cause multiple republish actions to occur if the customer performs the operation on multiple tabs.
- **Tab Disable Settings Loss** - Fixed a bug where 'disabling' a tab with the checkbox at the top would cause the settings to be lost if the Publisher was closed.
- **Webhook Duplication** - Fixed a bug where the alert webhooks may be duplicated during a setting import.
- **Auto-Enable Right-Click** - Fixed a bug where some unsupported right-click options might be enabled by the auto-enable product's rules.
- **Timeout Setting Read** - Fixed a bug where conflicting process notification timeout setting was not being read properly from settings.xml causing the setting to not apply.
