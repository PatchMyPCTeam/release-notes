# Publisher Preview Version 2.1.36.80

**Release Date:** 2025-07-09

## Overview

This preview release introduces migration capabilities from ConfigMgr to Patch My PC Cloud, adds a progress bar to the Publisher UI, and improves logging throughout the product. Several bugs related to Manage Conflicting Processes and UI filtering are resolved.

---

## Features

New capabilities and functionality added in this release.

- **ConfigMgr to Cloud Migration** - Migration from ConfigMgr to Patch My PC Cloud.
  - Read only for Enterprise+ customers.
  - Read/Write for Enterprise Premium customers.
- **Publisher Progress Bar** - Add a progress bar to the Publisher UI. It will display the current application or update being processed.

---

## Improvements

Enhancements to existing functionality.

- **Intune Scan Wizard UI** - Ensure the scrollbar on the Intune application scan wizard only shows when needed.
- **Script Signing Logging** - Improve the logging of PowerShell-based detection and requirement script signing.
- **Log File Location** - On new installs the PatchMyPC.log file is stored in the Logs subfolder of the installation directory.
- **Service Account Logging** - Write the name of the account used to run PatchMyPCService to the publishers log file.
- **Log File Extensions** - Amended the rolled over log file extension from .lo\_ to .log.
- **MCP User Logging** - Log username of active user when using Manage Conflicting Processes to kill processes.
- **PowerShell Logging** - Improved logging when ScriptRunner calls PowerShell scripts.

---

## Fixes

Bug fixes and issue resolutions.

- **MCP Application Closure** - Fixed a bug that caused [Manage Conflicting Processes](https://patchmypc.com/kb/manage-conflicting-processes-when-updating/) to fail to close running applications in some scenarios.
- **Webhook Logo URL** - Fixed the URL for the Patch My PC logo in webhook alerts.
- **Runtime Conflict** - Fixed a bug that would allow a configuration of maximum application runtime that conflicted with the Manage Conflicting Process notification runtime.
- **Intune Group Filtering** - Fixed a bug that caused the filtering of Intune groups to hang the UI
- **Log Retention** - Fixed a bug where rolled over log files were not cleaned up according to retention settings.
