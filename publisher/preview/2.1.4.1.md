# Publisher Preview Version 2.1.4.1

**Release Date:** 2022-05-19

## Overview

This preview release adds server name to upgrade notifications, improves desktop shortcut deletion for user-based apps, and fixes several bugs related to Intune tenants, MST forms, and SUP sync.

---

## Features

New capabilities and functionality added in this release.

- **Server Name in Notifications** - Include server name in Publisher upgrade notification email and Teams/Slack notification.
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-2136)*

---

## Improvements

Enhancements to existing functionality.

- **User Desktop Shortcuts** - The [Delete desktop shortcutâ€¦](https://patchmypc.com/custom-options-available-for-third-party-updates-and-applications#delete-shortcut) right-click option now removes shortcuts from the user desktop for user-based applications.
- **SMS Provider Highlight** - The SMS provider button is now highlighted in the WSUS options if the SMS provider is not configured.

---

## Fixes

Bug fixes and issue resolutions.

- **Empty Intune Tenant** - Fixed a bug where an empty Intune tenant is written to settings causing errors when the Publisher attempts to query the invalid tenant.
- **MST Form Layout** - Fixed the layout of the [Add MST transformation file](https://patchmypc.com/custom-options-available-for-third-party-updates-and-applications#mst-transform) form.
- **DP Group Query** - Fixed a bug where we may fail to query for Distribution Point groups if the name or description is DBNull instead of an empty string.
- **SUP Sync** - Fixed a bug where the ConfigMgr SUP sync would not start after a Patch My PC Sync if only updates using local content were published.
- **Intune Auto-Publishing** - Fixed a bug where the Intune auto-publishing may fail in some cases when right-click options are configured.
