# Publisher Preview 1.8.4.3 - 2020-08-14

## Overview

This preview release includes features and improvements for the Patch My PC Publisher.

## Features

- [You can now configure Intune Update descriptions. The description can include environment variables that will be replaced at publish time such as %ProductName%](https://patchmypc.com/intune-update-catalog-and-update-publishing-options#Intune-Update-Descriptions)
- [Added support for Anonymous SMTP. This is helpful when using SMTP Relay that does not require authentication](https://patchmypc.com/email-and-teams-notifications#anonymous-SMTP)

## Improvements

- [When the SMTP checkbox is enabled, a "Send Test Email" button will now be displayed](https://patchmypc.com/email-and-teams-notifications#send-test-email)
- Intune Win32 Apps and Intune Updates are now counted separately when displaying the enabled product count
