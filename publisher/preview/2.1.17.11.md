# Publisher Preview Version 2.1.17.11

**Release Date:** 2023-09-14

## Overview

This preview release fixes the 'Change Visibility' option for WSUS updates when the WSUS database has a custom name.

---

## Fixes

Bug fixes and issue resolutions.

- **WSUS Visibility with Custom DB** - Fixed a bug where the 'Change Visibility' option for WSUS updates would not work if the WSUS DB is called something other than SUSDB.
