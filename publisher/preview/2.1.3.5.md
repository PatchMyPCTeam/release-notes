# Publisher Preview Version 2.1.3.5

**Release Date:** 2022-04-27

## Overview

This preview release introduces certificate authentication for Azure App Registration, adds the ability to pause product updates, and enables per-tenant branding for Manage Conflicting Process in multi-tenant environments.

---

## Features

New capabilities and functionality added in this release.

- **Certificate Authentication** - Certificate Authentication for Azure App Registration
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1540)*
- **Pause Product Updates** - Option to pause creation of updates or applications for specifics products
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1554)*
  - Requires Enterprise Plus
- **Per-Tenant MCP Branding** - Allow per-tenant branding for Manage Conflicting Process
  - Applies to Multi-tenant private preview
