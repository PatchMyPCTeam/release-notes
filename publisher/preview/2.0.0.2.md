# Publisher Preview 2.0.0.2 - 2021-02-24

## Overview

This preview release includes features, improvements, and fixes for the Patch My PC Publisher.

## Features

- Added new options to allow you to exclude specific custom folders from being used on software updates or applications

## Improvements

- Improved the handling of distribution point status for updates, applications, and packages
- We now have much more logging around what distribution point groups are selected for each product

## Fixes

- Fixed an issue where application sources would not be distributed to distribution points
- Fixed an issue where the status of certain distributions was not properly detected
