# Publisher Preview Version 2.1.17.27

**Release Date:** 2023-09-29

## Overview

This preview release changes the OK button to 'Save and Close' for better clarity and fixes a race condition when multiple threads access settings components.

---

## Improvements

Enhancements to existing functionality.

- **Save and Close Button** - OK button has been changed to 'Save and Close'
  - Open to feedback on this change. We have received a fair number of reports that it is unclear the 'OK' button will close the UI.

---

## Fixes

Bug fixes and issue resolutions.

- **Settings Thread Safety** - Fixed a bug where multiple threads could access some components of settings at the same time, causing a race condition.
