# Publisher Preview Version 2.1.24.7

**Release Date:** 2024-06-14

## Overview

This preview release introduces the ability to manage return codes for applications and updates, improves binary download handling with temporary staging, and provides more contextual Manage Conflicting Processes tooltips.

---

## Features

New capabilities and functionality added in this release.

- **Manage Return Codes** - Manage return codes
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-556)*

---

## Improvements

Enhancements to existing functionality.

- **Staging Directory** - Use a temporary staging directory for binary downloads when processing ConfigMgr applications
- **Cloud Connection Handling** - Improved handling of the cloud connection.
- **Content Resolution** - Improved how ConfigMgr content is resolved to ensure we choose non-retained applications if present.
- **Contextual MCP Tooltips** - The tooltip and icon for Manage Conflicting Processes suggested configurations are now more contextual.
  - If configured to Kill or Notify, the tooltip and icon are removed
  - If configured to Skip, the icon is shown, and the tooltip says, "Manage Conflicting Processes is recommended for this product and will be configured to Skip by default"
  - If configured to 'Perform the installation,' the icon is shown, and the tooltip says, "Manage Conflicting Processes is recommended for this product but is currently not configured"

---

## Fixes

Bug fixes and issue resolutions.

- **Prevent Start Feature** - Fixed a bug that caused the 'Prevent the end-user from opening an application while the application is updating' feature to not work for ConfigMgr or Intune applications in some instances.
