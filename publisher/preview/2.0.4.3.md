# Publisher Preview Version 2.0.4.3

**Release Date:** 2021-06-02

## Overview

This preview release adds the ability to override manual Intune assignment changes on sync, displays application titles in the Manage Assignments wizard, and makes notifications presentation mode aware.

---

## Features

New capabilities and functionality added in this release.

- **Override Assignment Changes** - Add option to update Intune assignments on sync.
  - New checkbox at the bottom of the Manage Assignments wizard to 'Override manual assignment changes…'
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1297)*
- **Assignment Title** - Add title of application or update to the Manage Assignments wizard.
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1420)*
- **Presentation Mode Awareness** - Notifications are presentation mode aware
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1248)*

---

## Improvements

Enhancements to existing functionality.

- **Applicability Display** - When a product is double-clicked in the [show package info](https://patchmypc.com/custom-options-available-for-third-party-updates-and-applications#PackageInfo) tool the applicability rule will be shown.
- **Configuration Messages** - Improve the message which displays when an incorrect configuration is saved.
- **User-Based Prep** - Code changes in preparation for user-based applications.

---

## Fixes

Bug fixes and issue resolutions.

- **Required App MCP** - Fixed a bug where the Manage Conflicting Process window would not appear when a ConfigMgr application was deployed as required and the checkbox for 'Allow user interaction…' was not checked.
- **Scope Tags Sync** - Fixed a bug where Intune role scope tags would not be updated on sync for Intune Updates.
- **Proxy for Intune** - Fixed a bug where the configured proxy may not be used for the Intune connection during publisher sync.
