# Publisher Preview Version 2.1.36.90

**Release Date:** 2025-07-16

## Overview

This preview release adds support for ECDSA code signing certificates, introduces a product selection filter, modernizes UI icons, and fixes a critical ScriptRunner issue with MSI/MSP installer parameters introduced in version 2.1.36.80.

---

## Features

New capabilities and functionality added in this release.

- **ECDSA Certificate Support** - Add support for ECDSA code signing certificates.

---

## Improvements

Enhancements to existing functionality.

- **Product Selection Filter** - Add a filter button to only show products that are selected in the Publishers UI.
- **Entra ID Search Enhancement** - Trigger search in Entra ID Group Search window when pressing Enter key.
- **Certificate Import Warning** - Present a warning when importing a code signing certificate if ConfigMgr is set to automatically manage your code signing certificate.
- **Migration Performance** - Improve performance when Publisher is processing ConfigMgr apps for migration.
- **Intune Statistics** - Add additional fields, related to user install statistics, to the Intune Application Manager.
- **Modern Icons** - Modernised the icons in the Publishers UI.

---

## Fixes

Bug fixes and issue resolutions.

- **ScriptRunner MSI/MSP Parameters** - Fixed issue where ScriptRunner was not correctly appending MSI and MSP installers with /qn (Introduced in Preview release 2.1.36.80)

---

## Other

Additional changes and information for this release.

- **Remediation for 2.1.36.80 Issue** - If you published applications using version 2.1.36.80, you will need to re-publish any applications to ensure the correct installation parameters are applied. You can either update to this version (2.1.36.90) or roll back to the latest production release (2.1.36.0).
