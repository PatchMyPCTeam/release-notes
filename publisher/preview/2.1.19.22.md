# Publisher Preview Version 2.1.19.22

**Release Date:** 2024-02-02

## Overview

This preview release improves custom product querying from the cloud, adds EU region support for cloud connections, and fixes bugs related to tab copying and .intunewin file handling.

---

## Improvements

Enhancements to existing functionality.

- **Custom Products Query** - Improve how the list of custom products is queried from the cloud.
- **EU Region Support** - Improve the cloud connection flow to account for the EU region.
- **UI and Logging** - Various typo corrections in the UI and in the log.
- **Retention Logging** - Improve logging for retained ConfigMgr applications.
- **Exception Logging** - Expand out aggregate exceptions when they are logged.
- **Cloud UX Improvements** - Improved user experience both for a disconnected cloud configuration and an empty custom app list.

---

## Fixes

Bug fixes and issue resolutions.

- **EU Cloud Connection** - Fixed a bug that made connecting to an EU Patch My PC cloud customer inconsistent.
- **Tab Copy with Custom Apps** - Fixed a bug causing the copy between tab options not to work as expected when custom applications are configured.
- **ESP Association Copying** - Fixed a bug where ESP associations would copy between the Intune Apps and Intune Updates tab.
- **Large IntuneWin Files** - Improved how .intunewin files are handled to ensure we can process large files.
