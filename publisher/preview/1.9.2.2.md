# Publisher Preview 1.9.2.2 - 2020-10-08

## Overview

This preview release includes features and improvements for the Patch My PC Publisher.

## Features

- [The Publisher will now use the Configuration Manager client's log directory rather than always defaulting to 'C:\Windows\CCM\Logs'. Based on Aha Idea: PATCHMYPC-I-774](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-774)

## Improvements

- Improved ScriptRunner logging in various areas such as where the uninstall string is found, or when using the Kill Process feature
