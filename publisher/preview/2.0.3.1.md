# Publisher Preview Version 2.0.3.1

**Release Date:** 2021-04-22

## Overview

This preview release adds support for future Home Lab subscriptions, introduces a new context menu in the Intune Application Manager, and improves logging capabilities.

---

## Features

New capabilities and functionality added in this release.

- **Home Lab Subscription** - Added support to add a future Home Lab subscription
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1213)*
- **App Manager Context Menu** - Added a new context menu in the Intune Application Manager

---

## Improvements

Enhancements to existing functionality.

- **Enhanced Logging** - Improved logging
