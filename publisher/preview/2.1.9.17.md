# Publisher Preview Version 2.1.9.17

**Release Date:** 2022-10-27

## Overview

This preview release adds the ability to limit Intune upload threads, improves proxy configuration logging, and fixes bugs related to proxy usage and code signing validation.

---

## Improvements

Enhancements to existing functionality.

- **Upload Thread Limiting** - Add the ability to limit the number of threads used during the upload of Intune packages.
  - [Documentation](https://patchmypc.com/advanced-configurations-available-using-the-registry-for-patch-my-pcs-publishing-service#AzureUploaderMaxThreadCount)
- **Proxy Logging** - Adding more logging around proxy configuration and failures.
- **Desktop Shortcut Logging** - Scriptrunner will now log out a comma-separated list of all public desktop shortcuts if the installing product is configured to delete desktop shortcuts.
  - This is to help troubleshoot when icons for applications are not deleted.

---

## Fixes

Bug fixes and issue resolutions.

- **Proxy Configuration** - Fixed a bug where some network operations would not use the configured proxy.
- **Code Signing Validation** - Fixed a bug where the code signing of ConfigMgr detection scripts may fail to validate the digital signature on the endpoint.
