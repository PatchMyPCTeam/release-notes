# Publisher Preview 1.5.2.7 - 2019-09-19

## Overview

This preview release includes fixes for the Patch My PC Publisher.

## Fixes

- Fixed an issue where application detection method scripts may fail on devices with PowerShell version 2. The following output error would be logged to appdiscovery.log: Unexpected token '.0' in expression or statement. CategoryInfo : ParserError: (.0:String) [], ParseException. FullyQualifiedErrorId : UnexpectedToken. CScriptHandler::DiscoverApp failed (0x87d00327). Deployment type detection failed with error 0x87d00327
- The select application folder dialog may crash when SmsProvider name is empty are unreachable
- The select application folder dialog may try to connect to SMSProvider with a username and password even if the checkbox is unchecked
