# Publisher Preview Version 2.1.3.3

**Release Date:** 2022-02-23

## Overview

This preview release changes CSV file encoding to UTF-8 and fixes bugs related to Intune ADR and process name matching in Manage Conflicting Processes.

---

## Improvements

Enhancements to existing functionality.

- **UTF-8 CSV Files** - CSV files are now saved with UTF-8 formatting.

---

## Fixes

Bug fixes and issue resolutions.

- **Intune ADR Publishing** - Fixed a bug where Intune ADR (private preview) would publish both an Application and an Update.
- **Process Name Matching** - Fixed a bug where we might fail to match a running process with Manage Conflicting Processes if the case of the process name did not match.
