# Publisher Preview 1.9.9.4 - 2021-01-26

## Overview

This preview release includes features, improvements, and fixes for the Patch My PC Publisher.

## Features

- [You can now configure software update supersedence to use the "Immediately expire a superseded software update" option if desired. Based on Aha Idea: PATCHMYPC-I-314](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-314)

## Improvements

- Improved the XML catalog parsing logic to better detect issues when a product is configured in an unsupported way

## Fixes

- Fixed an issue where you may receive an error if the last/only product is disabled in the enabled products list
