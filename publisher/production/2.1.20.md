# Publisher Version 2.1.20 - Production

**Release Date:** 2024-02-21

## Overview

This release introduces Custom Apps functionality (in public preview), allowing customers to manage their own applications through Patch My PC Cloud. Improved cloud connection flow supports EU region, and memory usage is optimized for large .intunewin file uploads.

---

## Features

New capabilities and functionality added in this release.

- **Custom Apps** - Ability to create and manage custom applications through Patch My PC Cloud.
  - *[Requested by users](https://patchmypc.aha.io/ideas/PATCHMYPC-I-1303)*
  - [Documentation](https://patchmypc.com/ui-doc-publisher-cloud)

---

## Improvements

Enhancements to existing functionality.

- **Custom Product Queries** - Improve how the list of custom products is queried from the cloud.
- **EU Region Support** - Improve the cloud connection flow to account for the EU region.
- **UI and Logging Corrections** - Various typo corrections in the UI and in the log.
- **ConfigMgr Retention Logging** - Improve logging for retained ConfigMgr applications.
- **Exception Logging** - Expand out aggregate exceptions when they are logged.
- **User Experience** - Improved user experience both for a disconnected cloud configuration and an empty custom app list.
- **Memory Optimization** - Use less memory when uploading .intunewin files to Intune.
- **ConfigApi Handling** - Improve the user experience when the ConfigApi is not available.
  - The UI can now start without the ConfigApi being available. A popup message will still appear indicating it cannot be reached.
  - When this is not available, the cloud features of the Publisher will be disabled.

---

## Fixes

Bug fixes and issue resolutions.

- **EU Cloud Connection** - Fixed a bug that made connecting to an EU Patch My PC cloud customer inconsistent.
- **Copy Between Tabs** - Fixed a bug causing the copy between tab options not to work as expected when custom applications are configured.
- **ESP Associations** - Fixed a bug where ESP associations would copy between the Intune Apps and Intune Updates tab.
- **Intunewin File Handling** - Improved how .intunewin files are handled to ensure we can process large files.
- **Right-Click Options** - Fixed a bug that caused some right-click options to not clear their state as expected for custom apps.
- **Apply Button** - Editing Manage Conflicting Process options did not light up the Apply button.
- **Upload Retry** - Fixed a bug that caused the Publisher to fail to retry when uploading chunks to Azure.
- **Hash Comparison** - Fixed a bug where a custom application download may compare hash against an older version of the custom application's hash.
- **Treeview Background** - Fixed a bug that caused the treeview to have a yellow background, as if no products are selected, even when products were selected.
