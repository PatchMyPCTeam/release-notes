# Publisher Preview Version 2.1.6.35

**Release Date:** 2022-09-23

## Overview

This preview release allows the same Azure group to be assigned multiple times, enables passing variables into pre/post-scripts, adds content extraction capabilities, and allows conflicting processes to be configured for any app.

> **Note:** Did I miss 33 preview builds???? No, you did not. Patch My PC has started to increment version numbers automatically as changes are reviewed and merged. The result is preview builds having multiple internal builds before a public preview is released.

---

## Features

New capabilities and functionality added in this release.

- **Multiple Azure Group Assignments** - Allow the same Azure group to be assigned multiple times for Intune assignments. This allows a group to be used as both an include, and an exclude.
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-2322)*
- **Script Variables** - Pass variables into pre and post-scripts.
  - Note: the %ProductName% and %VendorName% variables are Base64 encoded when they are passed to the pre and post-scripts. It will need to be decoded. Patch My PC will provide a sample PowerShell snippet to decode the resulting parameter.
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1348)*
- **ConfigMgr Content Extraction** - Extract content from ConfigMgr applications.
- **WSUS Content Extraction** - Extract content from WSUS updates.
- **Universal Conflicting Processes** - Allow any app to have 'conflicting processes' configured
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1699)*

---

## Improvements

Enhancements to existing functionality.

- **Child Form Positioning** - Improved child-form handling in some cases, so they now open in the center of the parent form.
- **Multi-Selection Forms** - Multi-selection views, such as selecting application scopes or categories, now use a consistent form that allows filtering.

---

## Fixes

Bug fixes and issue resolutions.

- **Graph Batch URL** - Fixed a bug where the wrong URL was used for Microsoft Graph batch requests in some cases.
- **MCP Button Sizing** - Fixed a bug where the buttons in Managed Conflicting Process may not fit the text in some translations.
- **ConfigMgr App Window** - Fixed a bug where the ConfigMgr app options window is not resizable.
- **MCP Organization Name** - Fixed a bug where the Manage Conflicting Process Organization Name would not be set when a ConfigMgr application was revised.
- **Settings Save** - Fixed a bug where settings could not be saved if the internet was unreachable.
