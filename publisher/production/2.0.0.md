# Publisher Version 2.0.0 - Production

**Release Date:** 2021-01-29

## Overview

This major release introduces Intune update support, enabling automatic publishing of third-party application updates as Win32 apps in Microsoft Intune. This complements existing ConfigMgr functionality and expands the Publisher's capabilities to modern cloud-based device management.

---

## Features

New capabilities and functionality added in this release.

- **Intune Update Support** - The Publisher now supports publishing third-party application updates to Microsoft Intune as Win32 apps. This feature enables automatic update management for Intune-managed devices, similar to the existing ConfigMgr update functionality.

---

## Other

Additional changes and information for this release.

- **Major Version Release** - This is a major version release (2.0.0) that expands the Publisher's scope beyond Configuration Manager to include Microsoft Intune as a supported management platform.
- **Update Automation** - Intune updates leverage the same catalog and automation capabilities as ConfigMgr updates, ensuring consistent third-party application management across both platforms.
- **Cloud Integration** - This release marks the beginning of enhanced cloud service integration for the Patch My PC Publisher platform.
