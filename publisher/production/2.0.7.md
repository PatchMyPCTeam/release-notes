# Publisher Version 2.0.7 - Production

**Release Date:** 2021-06-30

## Overview

This release adds scheduling for the Modify Updates Wizard cleanup, fills out disk space requirements for Intune apps, and improves the Intune App Manager with default filtering. Right-click copy functionality is added to the Package Info window.

---

## Features

New capabilities and functionality added in this release.

- **Scheduled Update Cleanup** - Allow the [scheduling of the Modify Updates Wizard for Update Cleanup (unreferencedpackagefolders)](https://patchmypc.com/clean-up-third-party-updates-from-the-wsus-updateservicespackages-folder#AutoCleanUpdateServicesPackages).
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-797)*
- **Disk Space Required** - Fill out 'Disk Space Required' for Intune apps.
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1466)*
- **App Manager Filtering** - Intune app manager will filter by PMPC published apps by default, providing a drop-down to select non-PMPC or all apps.
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1460)*
- **Copy to Clipboard** - Within the '[Show Package Info](https://patchmypc.com/custom-options-available-for-third-party-updates-and-applications#PackageInfo)' window, you can now right-click on any cell to copy the cell data or the row data to your clipboard.
  - *[Requested by users](https://ideas.patchmypc.com/ideas/PATCHMYPC-I-1468)*

---

## Fixes

Bug fixes and issue resolutions.

- **CMTrace Compatibility** - Further improvements to CM log format for culture compatibility.
